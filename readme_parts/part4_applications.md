## 应用场景

本工具广泛应用于各类效率评价和生产率分析的研究中，以下列举8个典型应用领域：

### 1. 学术研究 — 经济学效率评价

- **研究主题**：全要素生产率（TFP）测算与分解
- **投入变量**：劳动力、资本存量
- **期望产出**：GDP
- **适用指数**：FGLR1992_C（最经典的Malmquist分解）
- **参考方向**：宏观经济学、发展经济学领域的效率研究

### 2. 论文写作 — 毕业论文 / 期刊论文

- **研究主题**：某行业/区域的效率变动分析
- **典型场景**：硕士/博士毕业论文的核心实证部分
- **优势**：一键生成8个工作表的结果，可直接用于论文表格
- **适用指数**：根据研究深度选择，Zofio2007分解最完整

### 3. 区域经济 — 省际/市际全要素生产率

- **研究主题**：中国30个省份全要素生产率变动研究
- **投入变量**：从业人员、资本存量、能源消费
- **期望产出**：地区GDP
- **非期望产出**：CO2排放、废水排放
- **适用指数**：PL2005_C（全局Malmquist，适合跨期比较）
- **论文标题示例**：《中国省际绿色全要素生产率测度及空间差异分析》

### 4. 产业分析 — 行业/企业效率比较

- **研究主题**：制造业细分行业效率评价
- **投入变量**：固定资产、从业人员、研发投入
- **期望产出**：工业总产值、利润总额
- **非期望产出**：工业三废排放
- **适用指数**：FGLR1994（三重分解，区分纯技术和规模效率）
- **论文标题示例**：《中国制造业绿色全要素生产率的动态演进》

### 5. 环境评价 — 绿色经济效率

- **研究主题**：考虑环境污染的绿色经济效率测评
- **投入变量**：劳动力、资本、能源
- **期望产出**：GDP、专利数
- **非期望产出**：SO2排放、烟粉尘排放
- **适用指数**：必须设置 `undesirable=1`，推荐PL2005
- **论文标题示例**：《长江经济带绿色全要素生产率测度及驱动因素》

### 6. 公共部门 — 医院/学校/政府绩效评估

- **研究主题**：公立医院运营效率评价
- **投入变量**：床位数、医护人员数、医疗设备价值
- **期望产出**：门诊量、住院量、手术量
- **非期望产出**：医疗事故数（可选）
- **适用指数**：FGLR1992_V（VRS假设更适合公共部门）
- **论文标题示例**：《基于SBM-Malmquist的公立医院效率动态评价》

### 7. 金融领域 — 银行/保险公司效率

- **研究主题**：商业银行经营效率分析
- **投入变量**：员工人数、固定资产、营业支出
- **期望产出**：贷款总额、净利润、中间业务收入
- **适用指数**：FGLR1994（区分规模效率和纯技术效率）
- **论文标题示例**：《中国上市商业银行全要素生产率变动研究》

### 8. 农业经济 — 农业生产效率

- **研究主题**：农业全要素生产率测算
- **投入变量**：播种面积、农业劳动力、化肥施用量
- **期望产出**：农业总产值、粮食产量
- **非期望产出**：农业面源污染（化肥流失量）
- **适用指数**：PL2005_C（适合长期趋势分析）
- **论文标题示例**：《中国农业绿色全要素生产率测度及收敛性分析》

### 如何选择合适的分解方法？

| 研究需求 | 推荐工作表 | 理由 |
|----------|------------|------|
| 基础效率评价 | sbmeffs | 查看原始效率值 |
| 经典TFP分析 | FGLR1992_C | 最常用的分解方法 |
| 需区分规模效率 | FGLR1994 | 三重分解 |
| 需完整分解 | Zofio2007 | 最详细的四重分解 |
| 避免技术退步矛盾 | PL2005_C | 全局前沿方法 |
| 公共部门/微观分析 | FGLR1992_V | VRS假设更合理 |
